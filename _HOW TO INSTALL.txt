This is a 2-step process because you need to change the default MIDI file that Rekordbox looks at for the controller to unlock custom MIDI assignments.
Basically, you first overwrite a default MIDI file in your Rekordbox system folder to remove the "Read-Only" restriction.

Note: Rekordbox may still show warning/error messages upon assignment, but the updates will apply successfully.

S T E P 1:
Copy the "DDJ-GRV6.midi.csv" file:
- FROM folder: "1 DEFAULT MIDI MAP TO COPY TO REKORDBOX SYSTEM"
- TO your Rekordbox directory's MidiMappings folder, typically:
  C:\Program Files\Pioneer\rekordbox 7.x.x\MidiMappings\
  or
  C:\Program Files\rekordbox\rekordbox 7.x.x\MidiMappings\

S T E P 2:
Open Rekordbox, click the "MIDI" button in the top right of the screen.
Import the custom mapping CSV file:
- For Standard Custom Map (Recommended):
  Import from folder "2 CUSTOM MIDI MAP"
- For Deluxe Custom Map:
  Import from folder "3 LEXS DELUXE MIDI MAPS"

Restart Rekordbox after completing these steps.
