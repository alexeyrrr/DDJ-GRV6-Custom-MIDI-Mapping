This is a 2 step process becayse you need to change the default Midi file that Rekordbox looks at for the controller to allow for MIDI assigment. 
Basically you first upload a default midi file to remove the "RO" (Read Only) requirement. 
Note: it may still show errors on assignment, but they will update.

S T E P 1:
Copy file: From folder "1 DEFAULT MIDI FILE TO COPY TO Key Mappings" the file named "DDJ-GRV6.midi.csv" to C:\Program Files\rekordbox\rekordbox 7.2.14\MidiMappings


S T E P 2:
Open Rekordbox, go to Midi config at the top right of the screen.
