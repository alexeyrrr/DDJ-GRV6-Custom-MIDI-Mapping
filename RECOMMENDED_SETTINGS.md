# Recommended Rekordbox Settings

Direct and recommended settings for the Pioneer DDJ-GRV6 custom mappings.

## 🎛️ Merge FX
*   **Recommendation:** Use **Filter Echo** as your primary (and only) Merge FX.
*   **Setup:** Open the FX panel using the icon on the left in Rekordbox or use the dedicated open FX panel button as part of the mapping.  Go to Preferences → **Controller** → Change the effects for MFX 1-4. I prefer just a simple echo filter the effect for both MFX1 & 2.

## 📊 Beat Jump Pad Setup
*   **Page Display:** Set the Beat Jump page display to show `+/- 16` and `32` beats in Rekordbox to match the repurposed PAD5–PAD8 layout on the controller.


## 🎙️ Stem Isolation
*   **Note:** While the funtionality in the deluxe mapping is technically non-destructive, it can get very confusing if someone is expecting native functionality. 
*   In short, to activate stem iso mode for channel 1, you press SHIFT + CUE on channel 3. Symmetrically, to activate stem iso mode for channel 2, you press SHIFT + CUE on channel 4.
*   This allows for constant hands-on control of both regular EQs and for stems for channels 1 and 2. 

## 🎧 Audio Routing
*   **External Soundcards:** If using an external soundcard instead of the internal GRV6 output, check Preferences → **Audio** → **Output Channels** to ensure routing is correct. Physical headphone mix and volume knobs will control software levels.
